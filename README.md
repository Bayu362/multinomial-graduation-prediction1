## 📌 Featured Research

### Multinomial Logistic Regression Modeling for Student Graduation Timeliness
> Undergraduate Thesis — Mathematics Study Program, UIN Malang · 2025

Built a predictive classification model to identify students at risk of delayed graduation across three categories: **On Time** (7–8 semesters), **Late** (9–10 semesters), and **Critical** (≥11 semesters).

**Dataset:** 331 students (cohorts 2018–2021), 7 predictor variables

| Metric | Value |
|---|---|
| Overall Accuracy | 64.18% |
| F1-Score — On Time | 75.55% |
| F1-Score — Late | 42.10% |
| F1-Score — Critical | 28.57% |
| Micro F1-Score | 64.18% |

**Key Findings:**
- **GPA** is the sole significant predictor in Logit 1 (Late vs On Time), coeff. −1.186
- **GPA** and **Credit Completion Ratio** are both significant in Logit 2 (Critical vs On Time), coeffs. −0.945 and −0.780
- Both significant predictors negatively associate with the probability of delayed graduation
- Non-academic factors (gender, school type, admission pathway, funding, region) were not statistically significant in the final model

**Methodology Pipeline:**

```
Data Collection → Cleaning & Dummy Coding → Multicollinearity Check (VIF)
    → Z-score Standardization → Stratified Train-Test Split
        → MLE Parameter Estimation → Significance Testing (LRT + Wald)
            → Goodness-of-Fit → Confusion Matrix Evaluation → Interpretation
```

**Predictor Variables:** GPA · Gender · School Type · Funding Type · Admission Pathway · Region of Origin · Credit Completion Ratio
